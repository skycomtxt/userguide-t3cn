# 参数设置

### 1. 熔接模式

![](../../.gitbook/assets/P2-1.operation-mode-cn-480-272.png)

**功能简介：**用于选择熔接模式。

| 熔接模式 | 功能说明              |
| ---- | ----------------- |
| 全自动  | 关闭防风罩后自动开始熔接      |
| 自动   | 关闭防风罩后，按“熔接键”开始熔接 |
| 手动   | 手动操作结构完成推进和对芯操作   |
